# Houseace Frontend
[Here](https://www.houseace.com.au/) is original WordPress website.
## Getting Started

First, run the development server:

```bash
yarn install

# Database
yarn migrate
yarn seed

# Dev
yarn dev
# Build
yarn build
# Start
yarn start
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

```bash
# Husky install
yarn postinstall
# Eslint fix
yarn lintfix
# Type check
yarn typecheck
```
